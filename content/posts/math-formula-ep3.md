[지난 편](/posts/math-formula-ep2/)에서 수학 실력의 다섯 가지 톱니바퀴를 봤습니다.

오늘은 이것들이 **서로 어떤 관계**인지 정리합니다.

---

## 다섯 개는 평등하지 않습니다

"다 중요하다"는 맞는데, **같은 방식으로 중요한 건 아닙니다.**

비유를 하나 들어볼게요.

### 전기 회로

집에 전기가 들어오는 과정을 생각해보세요.

```
차단기 → 전선 → 전구 → 밝기조절기
```

- **차단기**가 내려가면? 아무것도 안 됩니다. 전선이 아무리 좋아도.
- **전선**이 하나 끊어지면? 다른 전선으로 어느 정도 버팁니다.
- **전구**가 없으면? 전기는 오는데 빛이 안 납니다.
- **밝기조절기**를 올리면? 같은 전구인데 더 밝아집니다.

수학 실력도 이런 구조입니다.

---

## 회로로 보는 수학 실력

```
시동(차단기) → 이해+계산(전선) → 전략(전구) → 점검(밝기조절기)
```

| 톱니바퀴 | 회로에서의 역할 | 없으면? |
|---|---|---|
| 시동 | 차단기 | **전체 꺼짐.** 나머지가 아무리 좋아도 소용없음 |
| 이해 | 전선 A | 느리지만 계산(전선 B)으로 버틸 수 있음 |
| 계산 | 전선 B | 느리지만 이해(전선 A)로 버틸 수 있음 |
| 전략 | 전구 | **빛이 안 남.** 지식이 있어도 문제에 적용 못 함 |
| 점검 | 밝기조절기 | 작동은 하는데, 가진 것보다 어둡게 나옴 |

하나씩 풀어볼게요.

---

## 시동 = 차단기

```
시동이 꺼지면 (= "수학 포기")

이해 = 90점이어도
계산 = 90점이어도
전략 = 90점이어도
점검 = 90점이어도

결과 = 0점
```

차단기가 내려가면 집 전체에 전기가 안 들어오죠. 시동이 꺼진 학생은 문제를 **읽지도 않습니다.** 나머지 능력이 아무리 높아도 발휘할 기회가 없어요.

반대로, 시동이 약하면(= "겨우 하는 중") 나머지 능력이 전부 깎여서 나옵니다. 차단기가 반만 올라간 것처럼요.

**그래서 시동은 곱셈입니다.** 0이면 전체가 0.

---

## 이해와 계산 = 병렬 전선

여기서 재미있는 게 나옵니다.

```
이해만 있는 학생:
  "이건 인수분해로 풀어야 해" (아는데)
  → 실제로 인수분해 하다가 부호 실수 → 틀림
  → 느리지만, 방향은 맞음

계산만 있는 학생:
  부호 실수 안 함, 계산 빠름 (잘하는데)
  → 근데 이게 인수분해인지 근의 공식인지 이해 못함
  → 기계적으로는 맞추는데, 응용에서 무너짐
```

핵심: **하나만 있어도 어느 정도 작동합니다.**

전선이 두 개 있는데 하나가 끊어져도 다른 하나로 전기가 흐르잖아요. 그래서 이해와 계산은 **덧셈**에 가깝습니다. 하나가 0이어도 전체가 0이 되진 않아요.

### 근데 둘 다 있으면?

```
이해만:  6점
계산만:  6점
둘 다:   6 + 6 + 보너스 = 15점?!
```

**둘 다 있으면 단순 합보다 큽니다.** 이해하면서 빠르게 푸는 사람은 응용문제까지 풀 수 있거든요. 이걸 **시너지**라고 합니다.

"아, 이래서 이 공식이 이렇게 생긴 거구나" — 이 순간이 시너지가 터지는 순간입니다. 계산하면서 이해가 깊어지고, 이해가 깊어지면서 계산이 빨라지는 선순환.

---

## 전략 = 전구

```
이해 ✅, 계산 ✅, 근데 전략 ❌

= 전기는 오는데 전구가 없는 상태
= 재료는 있는데 요리를 못 하는 상태
```

"배운 건 다 아는데, 이 문제에 뭘 써야 할지 모르겠어요."

이게 전략이 없는 상태입니다. 전기(지식)는 충분한데 빛(결과)이 안 나와요. **전략도 곱셈입니다.** 여기가 0이면 지식이 출력으로 연결되지 않아요.

---

## 점검 = 밝기조절기

점검은 좀 특별합니다.

```
점검 없이도 작동은 합니다.
  → 차단기 올라가 있고, 전선 연결되어 있고, 전구 있으면
  → 빛은 납니다.

근데 점검이 있으면?
  → 같은 전구가 더 밝게 빛납니다.
```

점검은 **없어도 풀 수는 있지만, 있으면 실력 이상을 발휘**하게 해줍니다.

같은 지식의 두 학생이 시험을 보면:
- 점검 없는 학생: 80점 (실수 + 시간 낭비)
- 점검 있는 학생: 95점 (실수 잡음 + 시간 배분)

같은 지식인데 15점 차이. 이게 밝기조절기의 효과입니다. **점검은 증폭기**예요.

---

## 전체 구조

정리하면 이렇습니다.

```
시동(차단기)                    → 꺼지면 전체 0
  ×
이해 + 계산 + 시너지(전선)      → 하나만 있어도 작동, 둘 다 있으면 보너스
  ×
전략(전구)                      → 없으면 출력 안 됨
  ×
점검(밝기조절기)                 → 없어도 되지만 있으면 증폭
```

이걸 공식으로 쓰면:

```
수학 실력 = 시동 × (이해 + 계산 + 시너지) × 전략 × (1 + 점검)
```

각 항의 의미:

| 항 | 의미 | 0이면? |
|---|---|---|
| 시동 | 곱셈 — 게이트 | 전체 0 |
| 이해 + 계산 | 덧셈 — 병렬 | 하나만 있어도 작동 |
| 시너지 | 보너스 — 둘 다 있을 때만 | 0이어도 작동, 있으면 점프 |
| 전략 | 곱셈 — 직렬 | 지식이 출력 안 됨 |
| (1 + 점검) | 증폭 — 부스터 | 1 (원래 실력 그대로) |

## 숫자로 보면

각 능력을 0~1 점으로 놓고 시뮬레이션해봅시다.

### 수포자

```
시동=0.1, 이해=0.3, 계산=0.5, 전략=0.4, 점검=0.2

= 0.1 × (0.3 + 0.5 + 보너스) × 0.4 × (1 + 0.2)
= 0.1 × 0.9 × 0.4 × 1.2
= 0.043

거의 바닥. 시동이 모든 걸 깎아버림.
```

### 노력파 (계산은 하는데 이해 약함)

```
시동=0.8, 이해=0.3, 계산=0.9, 전략=0.6, 점검=0.3

= 0.8 × (0.3 + 0.9 + 약간) × 0.6 × 1.3
= 0.8 × 1.3 × 0.6 × 1.3
= 0.81
```

### 이해파 (원리는 아는데 계산 느림)

```
시동=0.8, 이해=0.9, 계산=0.3, 전략=0.6, 점검=0.5

= 0.8 × (0.9 + 0.3 + 약간) × 0.6 × 1.5
= 0.8 × 1.3 × 0.6 × 1.5
= 0.94
```

### 균형형

```
시동=0.9, 이해=0.7, 계산=0.7, 전략=0.7, 점검=0.7

= 0.9 × (0.7 + 0.7 + 큰 보너스) × 0.7 × 1.7
= 0.9 × 1.7 × 0.7 × 1.7
= 1.82

시너지와 점검이 폭발적으로 작용.
```

---

## 이 공식이 말하는 것

```
1. 시동부터 고쳐라
   → 0.1이 곱해지면 나머지가 아무리 높아도 바닥

2. 이해와 계산, 둘 다 해라
   → 하나만으론 한계, 둘 다 있어야 시너지

3. 전략을 가르쳐라
   → 지식이 있어도 전략 없으면 시험에서 무너짐

4. 점검은 마지막 부스터
   → 실력이 높을수록 점검의 증폭 효과가 큼
   → (0.1에 1.5를 곱해봐야 0.15지만, 1.0에 1.5를 곱하면 1.5)
```
