[이전 편](/posts/math-formula-ep1/)에서 수학 실력이 하나가 아니라 **다섯 개의 톱니바퀴**라고 했습니다.

오늘은 그 다섯 개를 하나씩 뜯어봅니다.

---

## 1. 시동 — "해볼 마음이 있는가"

### 이건 실력이 아니라 전제조건입니다

카페를 생각해보세요.

> 문 앞에서 "여기 맛없을 것 같아"하고 돌아가는 사람에게는
> 아무리 맛있는 커피도 의미가 없습니다.

수학도 같습니다. "어차피 못해"라고 생각하면 문제를 **읽지도 않습니다**. 읽지 않으면 당연히 못 풀고, 못 풀면 "역시 못해"가 확인되고, 다음에도 안 읽습니다.

```
안 읽음 → 못 풀음 → "역시 못해" → 안 읽음 → ...
```

무한 루프입니다. 이걸 **악순환**이라고 합니다.

반대도 있어요.

```
풀어봄 → 하나 맞음 → "어 되네?" → 또 풀어봄 → ...
```

이게 **선순환**입니다.

### 핵심

시동은 지식의 문제가 아니라 **마음의 문제**입니다. 그리고 한번 꺼지면 다시 켜기가 아주 어렵습니다. 수포자의 진짜 문제는 대부분 여기에 있습니다.

---

## 2. 이해 — "왜 되는지 아는가"

### 외운 것과 이해한 것은 다릅니다

퀴즈 하나 내볼게요.

> 원의 넓이 = πr²
>
> 이거 왜 πr²인지 설명할 수 있나요?

대부분 못합니다. "그냥 공식이니까"라고 하죠.

근데 이해한 사람은 이렇게 설명합니다:

> "원을 피자처럼 아주 얇게 잘라서 펼치면 삼각형 비슷한 게 되거든요.
> 밑변이 원둘레의 반(πr)이고 높이가 r이니까,
> 삼각형 넓이 = ½ × πr × r = πr²/2... 아 아닌데,
> 위아래 합치면 πr × r = πr². 아 맞다!"

좀 더듬거려도 괜찮아요. 중요한 건 **"왜 이 공식이 이렇게 생겼는지" 설명하려고 시도할 수 있다**는 겁니다.

### 외운 사람 vs 이해한 사람

| 상황 | 외운 사람 | 이해한 사람 |
|---|---|---|
| πr² 계산 | ✅ 가능 | ✅ 가능 |
| "반지름이 2배면 넓이는?" | 🤔 다시 계산... | "r²이니까 4배" (즉답) |
| "타원의 넓이는?" | ❌ 새 공식 필요 | "πab겠네" (구조에서 유추) |
| 공식을 까먹었을 때 | ❌ 끝 | 🔄 유도해서 복구 |

이해한 사람은 **공식을 잊어버려도 다시 만들 수 있습니다**. 외운 사람은 까먹으면 끝이에요.

## 3. 계산 — "정확하고 빠르게 할 수 있는가"

### 이해와 계산은 다른 근육입니다

피아노를 생각해보세요.

> 악보를 읽을 줄 아는 것 (이해) ≠ 손가락이 빠르게 움직이는 것 (계산)

둘 다 필요합니다. 악보를 읽을 줄 알아도 손가락이 안 따라가면 연주를 못하고, 손가락이 빨라도 악보를 못 읽으면 새 곡을 못 칩니다.

수학도 마찬가지예요.

```
이해 = "이 문제는 인수분해를 해야 해" (아는 것)
계산 = 실제로 인수분해를 정확하고 빠르게 하는 것 (하는 것)
```

### "아는데 자꾸 틀려요"

이 말을 하는 학생은 2번(이해)은 있는데 3번(계산)이 부족한 겁니다. 부호를 빠뜨리거나, 분수 통분을 실수하거나, 전개를 잘못하거나.

이건 "더 이해해야 할 것"이 아니라 **"더 반복해야 할 것"**입니다. 아는 것과 몸에 익힌 것은 다른 영역이에요.

## 4. 전략 — "뭘 써야 하는지 아는가"

### 냉장고 앞에 선 요리사

이게 가장 과소평가되는 능력입니다.

> 냉장고에 재료는 가득합니다. 레시피도 다 알아요.
> 근데 "오늘 뭘 만들지?"를 결정 못하면 아무것도 못 만들어요.

시험에서 이런 경험 없나요?

> "분명히 배운 건데... 이 문제에 뭘 써야 하지?"

공식도 알고 계산도 할 수 있는데, **이 문제에 어떤 공식을 꺼내야 하는지** 모르는 겁니다.

### 왜 이런 일이 벌어지나

수학을 배울 때 보통 이렇게 합니다:

```
[미분 단원]
  미분 공식 배움 → 미분 문제 20개 풀음 → 미분 시험

[적분 단원]
  적분 공식 배움 → 적분 문제 20개 풀음 → 적분 시험
```

이러면 "미분 단원이니까 미분으로 풀면 되지"라는 생각이 들어요. **뭘 쓸지 고를 필요가 없었던 거예요.**

근데 시험에선?

```
1번: 미분? 적분?
2번: 확률? 경우의 수?
3번: 삼각함수? 지수?
```

전부 섞여 나옵니다. 그제서야 "뭘 써야 하지?"가 문제가 됩니다.

## 5. 점검 — "내가 지금 잘 하고 있는지 아는가"

### 내비게이션 없는 운전

길을 아는 것과 **지금 내가 맞는 길로 가고 있는지 아는 것**은 다릅니다.

내비게이션이 하는 일을 생각해보세요:

- "300m 앞에서 우회전" (다음 단계 안내)
- "경로를 이탈했습니다" (실수 감지)
- "더 빠른 경로가 있습니다" (전략 전환)

수학에서도 이 "내비게이션"이 있는 사람과 없는 사람의 차이가 큽니다.

### 같은 실력, 다른 결과

두 학생이 같은 시험을 봅니다. 지식도 비슷해요.

**학생 A** (내비게이션 없음):
```
1번 풀이 시작 → 5분째 안 풀림 → 계속 시도
→ 8분째 포기 → 2번으로 → 시간 부족 → 뒷문제 못 봄
```

**학생 B** (내비게이션 있음):
```
1번 풀이 시작 → 3분째 "이 방법 아닌 것 같다"
→ 일단 넘김 → 2~5번 빠르게 풀음 → 남은 시간에 1번 재도전
```

같은 실력인데 B가 점수가 높습니다. 차이는 **"내가 지금 잘 하고 있나?"를 감시하는 능력**이에요.

## 다섯 개의 관계

여기까지 읽으면 궁금해질 겁니다.

> "다섯 개 다 중요한 건 알겠는데,
> 그래서 이것들이 서로 어떤 관계야?
> 하나가 없으면 어떻게 돼?
> 뭐부터 키워야 해?"

단순히 "다 키우면 돼"가 아닙니다.

어떤 건 **없으면 전체가 무너지고** (시동),
어떤 건 **하나만 있어도 어느 정도 작동하고** (이해, 계산),
어떤 건 **있으면 실력 이상을 발휘하게 해줍니다** (점검).

이 관계를 하나의 구조로 정리하면 꽤 명쾌한 그림이 나옵니다.
