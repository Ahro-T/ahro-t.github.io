[이전 편](/posts/math-formula-ep3/)에서 수학 실력 공식을 봤습니다.

```
수학 실력 = 시동 × (이해 + 계산 + 시너지) × 전략 × (1 + 점검)
```

오늘은 **각 항을 어떻게 키우는지**, 그리고 **어떤 순서로 키워야 하는지** 정리합니다.

---

## 순서가 있습니다

아무거나 먼저 키우면 안 됩니다.

```
시동이 꺼진 학생에게 공식을 가르치면?    → 안 들음
이해가 없는 학생에게 반복 연습시키면?    → 기계적 암기, 곧 무너짐
지식이 없는 학생에게 전략을 가르치면?    → 전략을 골라도 실행할 재료가 없음
전략이 없는 학생에게 점검을 가르치면?    → "방법 바꿔야지" 해도 대안이 없음
```

아래가 충분해야 위가 작동합니다. 그래서 순서는:

```
Phase 1: 시동 켜기
Phase 2: 이해 + 계산 쌓기
Phase 3: 전략 장착하기
Phase 4: 점검 습관 들이기
```

---

## Phase 1: 시동 켜기

**목표: "이거 해볼 만하다"를 심는 것**

시동은 지식으로 켜지지 않습니다. 수학 공식을 아무리 알려줘도 "어차피 못해"라고 생각하면 안 들어요.

### 방법 1: 풀 수 있는 문제부터

```
❌ 첫 수업에 어려운 문제
   → 못 풀음 → "역시 못해" → 시동 꺼짐

✅ 80%는 풀 수 있는 문제로 시작
   → 풀음 → "어 되네?" → 시동 걸림
```

성공 경험이 시동을 켭니다. 학습 심리학에서는 이걸 **자기효능감**이라고 해요. "나도 할 수 있다"는 믿음.

### 방법 2: "왜 배우냐"에 답하기

> "이거 어디에 써요?"

이 질문에 "시험에 나와"라고 답하면 시동이 켜질까요?

```
❌ "시험에 나와"        → 시험 끝나면 끝
✅ "게임 확률 계산"     → "오 그런 거였어?"
✅ "이자율 복리 계산"   → "이거 알면 돈 관련 판단 가능"
✅ "알고리즘 면접"      → "이래서 코테에 수학 나오는구나"
```

수학이 **자기 인생과 연결된다**고 느끼면 시동이 켜집니다.

### 방법 3: 말을 바꾸기

```
❌ "이것도 못해?"      → 시동 꺼짐
❌ "넌 수학 재능이 없어" → 시동 영구 파괴

✅ "아직 안 된 거야"    → 가능성 열림
✅ "접근 방법이 좋았어"  → 과정에 대한 인정
```

"못해"가 아니라 "아직"이라는 한 단어가 시동을 살립니다.

---

## Phase 2: 이해 + 계산 쌓기

시동이 켜졌으면 이제 재료를 채웁니다.

### 이해를 키우는 법

#### 그림으로 보여주기

```
"미분은 접선의 기울기"

말로 하면:     🤷 "그래서요?"
그래프로 보면:  😮 "아, 이 점에서의 변화율이구나"
```

추상적인 걸 눈에 보이게 만드는 순간 이해가 점프합니다.

#### "왜?" 한 번만 더 묻기

```
학생: "음수 × 음수 = 양수예요"
교사: "왜?"
학생: "..."
교사: "방향을 생각해봐. 뒤로 가는 걸 뒤집으면?"
학생: "아... 앞으로 가는 거네!"
```

공식을 말한 후 **"왜?"를 한 번만 더** 묻는 습관. 이게 이해의 깊이를 만듭니다.

#### 여러 방식으로 같은 걸 보기

```
y = x²을 이해하는 4가지 방법:

  식:     y = x²
  그래프:  포물선
  표:     x=1→1, x=2→4, x=3→9
  말:     "x가 커질수록 y가 제곱으로 빠르게 커진다"
```

하나의 표현에 갇히면 응용이 안 됩니다. 여러 표현을 오가는 연습이 진짜 이해입니다.

### 계산을 키우는 법

#### 간격을 두고 반복

```
❌ 하루에 50문제 몰아풀기 → 다음날 까먹음

✅ 오늘 10문제 → 3일 뒤 5문제 → 1주 뒤 3문제
   → 까먹을 때쯤 다시 하면 장기기억으로 넘어감
```

이걸 **간격 반복**이라고 합니다. 벼락치기보다 3배 효과적이라는 연구 결과가 있어요.

#### 실수 유형 기록하기

```
"또 부호 틀렸네" → 이번 달 부호 실수 5회
"통분 실수"      → 이번 달 통분 실수 3회

→ 나의 약점: 부호 > 통분 > 전개
→ 부호 관련 문제만 집중 연습
```

뭉뚱그려 "계산 실수"라고 하지 말고, **유형별로 기록**하면 약점이 보입니다.

### 시너지 만들기

이해와 계산이 어느 정도 쌓였으면, 이 둘을 **연결**합니다.

```
인수분해를 하면서:
  "이 단계는 왜 하는 거지?"
  "곱해서 6, 더해서 5인 두 수... 이건 (x+2)(x+3)의 구조니까..."
```

계산하면서 "왜"를 붙이는 순간 시너지가 터집니다. 절차가 개념과 연결되는 순간이에요.

---

## Phase 3: 전략 장착하기

재료(이해+계산)가 쌓였으면, 이제 **"이 문제에 뭘 쓸지"** 고르는 능력을 키웁니다.

### 방법 1: 풀지 말고 분류만

```
문제 20개를 줍니다.

"풀지 마. 유형만 분류해."

  1번 → 이차방정식
  2번 → 확률
  3번 → 미분의 활용
  ...
```

풀이와 분류를 **분리해서 연습**합니다. 대부분의 학생은 풀이 연습만 하고 분류 연습은 안 해요. 그래서 시험에서 "뭘 써야 하지?"가 되는 겁니다.

### 방법 2: 유형을 섞어서 연습

```
❌ 미분 20문제 → 적분 20문제 → 확률 20문제
   (유형별 묶음 → 뭘 쓸지 고를 필요가 없음)

✅ 미분 → 확률 → 적분 → 미분 → 도형 → 확률
   (섞기 → 매번 "이건 뭘로 풀지?" 판단 필요)
```

처음엔 섞어서 푸는 게 더 어렵습니다. 당연해요 — 유형 판단까지 해야 하니까. 근데 이게 시험 상황이에요. 연습할 때 편하면 시험에서 어렵고, 연습할 때 어려우면 시험에서 편합니다.

### 방법 3: 패턴 카드 만들기

문제를 많이 풀다 보면 **"이런 단서가 보이면 이 방법"** 패턴이 생깁니다. 이걸 명시적으로 정리하면:

```
┌───────────────────────────────────────┐
│ 단서: "두 근의 합/곱"                  │
│ → 비에타 공식                         │
│                                       │
│ 단서: "모든 자연수 n에 대해"            │
│ → 수학적 귀납법                        │
│                                       │
│ 단서: "최대/최소"                      │
│ → 미분해서 0인 점 찾기                 │
└───────────────────────────────────────┘
```

이런 카드를 쌓아가는 것 = **패턴 라이브러리 구축**. 문제를 많이 푸는 것보다 이 라이브러리를 의식적으로 만드는 게 전략 능력에 직결됩니다.

---

## Phase 4: 점검 습관 들이기

마지막입니다. 이건 "가르치는" 게 아니라 **"습관을 심는"** 영역이에요.

### 3가지 자문 습관

문제를 풀 때 이 질문을 계속 자기에게 합니다:

```
시작 전:  "이건 어떤 유형이지? 어떤 방법으로 갈까?"  (계획)
풀면서:   "방향이 맞나? 계산이 복잡해지면 뭔가 잘못된 거 아냐?"  (감시)
풀고 나서: "답이 말이 되나? 더 빠른 방법은 없었나?"  (평가)
```

처음엔 어색합니다. 그래서 **카드를 책상에 놓고** 풀면 됩니다:

```
□ 지금 뭘 하고 있지?
□ 이 방향이 맞나?
□ 3분째 안 풀리면 방법을 바꿔라
```

외부 장치(카드)로 시작해서, 반복하면 내면의 습관이 됩니다.

### "막히는 것"이 정상이라는 것

점검 능력이 없는 학생은 **막히면 패닉**합니다. "나 못하나 보다."

점검 능력이 있는 학생은 **막히면 전환**합니다. "이 방법은 아닌 것 같으니 다른 걸로."

차이는 능력이 아니라 **"막히는 건 정상이다"라는 인식**입니다. 수학 잘하는 사람도 막힙니다. 다만 막혔을 때 **반응이 다를 뿐**이에요.

---

## 전체 요약

```
Phase 1: 시동
  "해볼 만하다"를 심는다
  → 풀 수 있는 문제, 실용 연결, 말 바꾸기

Phase 2: 이해 + 계산
  재료를 쌓는다
  → 그림/다중표현, 간격 반복, 실수 기록
  → 둘을 연결하면 시너지

Phase 3: 전략
  "뭘 쓸지" 고르는 힘
  → 분류 훈련, 섞어서 연습, 패턴 카드

Phase 4: 점검
  "잘 하고 있나" 감시 습관
  → 자문 카드, 막힘의 정상화
```

---

## 마지막으로

이 시리즈를 한 문장으로 줄이면 이겁니다.

> **수학을 못하는 이유는 하나가 아니고, 키우는 순서가 있다.**

"공식을 더 외워라"도 아니고, "문제를 더 풀어라"도 아닙니다. **어디가 약한지 찾고, 맞는 순서로 키우는 것.** 그게 전부입니다.

수학 실력은 재능이 아니라 구조입니다. 구조를 알면 고칠 수 있습니다.
